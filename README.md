# 项目搭建

## 使用Rsbuild框架创建项目
- cnpm create rsbuild@latest
- cnpm install react-router-dom@7.6.x
- cnpm install zustand ahooks@3.9.x
- cnpm install

## 核心库
- cnpm install @tauri-apps/api

## 初始化项目
