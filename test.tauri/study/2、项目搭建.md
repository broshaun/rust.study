## 创建项目
    create-react-app app
    cd app

## 初始化核心库
    cargo install tauri-cli
    cargo tauri init


## 前端组件库

#### 核心库访问接口
    cnpm install @tauri-apps/api

#### 代理插件
    cnpm install http-proxy-middleware --save

#### 路由库
    cnpm install react-router
    cnpm install react-router-dom

#### 样式库
    cnpm install semantic-ui-react semantic-ui-css

#### 提示框
    cnpm install react-hot-toast

#### 布局插件
    cnpm i react-grid-layout


## 执行调试
    cargo tauri dev

## 执行编译
    cargo tauri build
