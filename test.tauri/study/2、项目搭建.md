# Rsbuild v1.6.15构建
- npm  create rsbuild app -- --template react
- cd app

## 卸载当前版本（如果已安装更高版本）
- cnpm uninstall react react-dom react-scripts

## 安装 React 18 稳定版
- cnpm install react@18.x react-dom@18.x react-scripts@5.x react-router-dom@7.6.x
- cnpm install zustand
- cnpm install ahooks@3.9.x
- cnpm install


# 使用create-tauri-app 创建Tauri项目
- cargo install create-tauri-app --locked
- cargo create-tauri-app


# 如果有现有项目
## 安装 Tauri 的 CLI 工具
- cargo install tauri-cli
## 初始化 Tauri
- cargo tauri init






## 运行开发服务器
- cargo tauri dev
## 执行编译
- cargo tauri build


## 前端组件库
#### 核心库访问接口
- cnpm install @tauri-apps/api
